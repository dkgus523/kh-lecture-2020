# 4. 문제

HTML문서에서 영역을 구분할 때 사용하는 div태그와 span태그에 대해 설명하시오. (35점)

1. div 태그  
기본 값이 블록 속성. 수직 공간 분할. 수평으로 하나의 요소만 위치할 수 있음. 너비, 높이 존재    

2. span 태그  
기본 값이 인라인. 수평 공간 분할. 수평에 여러 요소가 위치할 수 있음. 너비, 높이 없음
